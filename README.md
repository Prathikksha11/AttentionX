#  AttentionX - AI Content Repurposer

Turn long videos into viral short clips automatically 

---

##  Features

-  Extracts audio from videos
-  AI-powered transcription using Whisper
-  Detects high-engagement moments
-  Generates multiple viral clips (<60s)
-  Converts to vertical format (Reels/Shorts)
-  Adds virality hooks + scoring

---

##  Tech Stack

- Streamlit
- OpenAI Whisper (offline)
- MoviePy
- Python

---

##   How it works

1. Upload a video  
2. AI transcribes content  
3. Detects engaging segments  
4. Generates multiple short clips  
5. Outputs ready-to-post reels  

---

##  Use Cases

- Content creators   
- Influencers 
- Podcasters   
- Marketing teams

##  Innovation

Instead of manual editing, AttentionX:
- Identifies viral-worthy segments
- Automatically formats for social media
- Reduces editing time from hours → seconds
